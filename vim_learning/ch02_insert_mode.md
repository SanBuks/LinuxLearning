# 第二章 编辑模式
## 1. 模式转换
- `i` 在光标前插入 `I` 在所在行第一个非空格前插入
- `a` 在光标后插入`A` 在所在行最后一个字符后插入
- `o` 在光标下一行处插入一行 `O`上一行处插入新的一行
- `r` 替换光标所在字符一次 `R` 一直替换光标所在字符直到退出
- `ctrl + c` 或者 `ctrl + [` 退出编辑模式

## 2. 编辑快捷键
- `ctrl + h`删除上一个字母
- `ctrl + w`删除上一个单词
- `ctrl + u`删除一行
- `ctrl + R + [寄存器名字]` 粘贴对应内容

## 3.关键词补齐
- `ctrl + n/p` 文件内容补齐 (也是选择方向键)
- `ctrl + f` 文件名补齐
- `ctrl +x + ctrl + o` 全能补齐
